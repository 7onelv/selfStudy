    Python数据分析04 数据模型

    本章节会涉及：

    1. 单因子和多因子线性回归（linear regression）
    2. 模型可视化和评估
    3. 多项式回归（polynomial regression）
    4. 回归评价指标 MSE R-squared
    5. 评估和决策制定

    模型在此的定义为通过分析一个或多个因变量，得出独立变量的一种方式或方法。通常来说因变量关联性越强，最后得出的独立变量准确性越高。

    线性回归 Linear Regression

    • 线性回归指通过一个因变量来测算目标变量
    • 多维线性回归指通过多个因变量来测算目标变量

    线性回归的通用公式为 y = b0 + b1x * b0: the intercept * b1: the slope

    通常，会将 x-value 和 y-value 分别储存在两个 matrix 当中，应用 python 中的 numpy。且 linear regression 很难实现完全、完整地 fit。这里引入 noise 的概念。测试人员会允许一定的误差，误差可能是正向的，也可能是负向的。且误差的分布通常也遵循正态分布，即大部分的误差接近于0，小部分的误差较为极端。

    上述提到的概念只仅限于一个独立变量。但如果有两个、三个独立变量呢？以两个独立变量为例，其公式会变为 y = b0 + b1x1 + b2x2。其示例图也会变为三维的，如下图。

    在 python 中应用线性回归

    一个变量的线性回归 

    # 导入模块
from sklearn.linear_model import LinearRegression
# 创建线性回归对象
lm=LinearRegression()
# 确定 x 和 y 的值
X = df[['highway-mpg']]
Y = df[['price']]
# 使用 lm.fit 训练模型
lm.fit(X,Y)
# 确定 b0, b1 的值
b0, b1 = lm.intercept_, lm.coef_
# 创建 yhat 结果集 len(x)=len(yhat)
Yhat = lm.predict(X)
# 应用测试 会返回 x [0:5] 对应的 y 值
Yhat[0:5]

    多个变量的线性回归 

    # 将多个变量储存在对象 Z 中
Z = df[['horsepower','curb-weight','engine-size','highway-mpg']]
lm.fit(Z, df['price'])
# 测算 yhat
Yhat = lm.predict(Z)
# 确定b0, b1, b2..的值
b0, b = lm.intercept_, lm.coef_
# 其中 lm.coef_ 的形式为array, 如 array([52.65, 4.70, 81.96, 33.58])

    其中，Y hat的定义为: the predicted value of y (the dependent variable) in a regression equation. It can also be considered to be the average value of the response variable. The regression equation is just the equation which models the data set. 即模型在不同独立变量情况下应用模型的测算值。

    多项式回归 Polynomial Regression

    多元回归为线性回归的一种特殊范畴，其在解释 curvilinear relationship 时非常有效。curvilinear relationship 指采用某个变量的平方值或 power 来作为 x-value。多元回归和 multiple linear regression 的不同在于前者是有了平方、立方等，而后者仅是有了多个独立变量。 * quadratic - squared degree 2 * cubic - 3rd order degree 3 * higher order

    另外一种方式是使用 Pipelines。Pipeline 可以有效简化模型建模的步骤。如下图，包括了 3 个步骤，数据正常化、多项式回归转换、线性回归模型，最后返回 yhat。

    在 python 中实现多项式回归

    f = np.polyfit(x,y,3)
# 使用 poly1d 转换为公式
p = np.poly1d(f)
print(p) 
print(np.ployfit(x, y, 3)) # array([-1.55663829e+00,  2.04754306e+02, -8.96543312e+03,  1.37923594e+05])

# 使用 preprocessing 库
from sklearn.preprocessing import PolynomialFeatures
# 创建一个 degree/order =2 的多元回归对象
pr = PolynomialFeatures(degree=2, include_bias=False)
Z_pr = pr.fit_transform(df[['horsepower','curb-weight']])
# 查看各种组合，第一个数据为样本值，第二个为 feature 值
Z_pr.shape # (201,4)

    ::使用 pipelines:: 

    from sklearn.preprocessing import PolynomialFeatures
from sklearn.linear_model import LinearRegression
from sklearn.preprocessing import StandardScaler

# 确定 pipelien 步骤
Input = [('scale',StandardScaler()), ('polynomial',PolynomialFeatures(degree=2)),('model',LinearRegression())]

# 创建 pipeline
pipe = Pipeline(Input)

    模型测试和评估

    模型可视化评估 Model Evaluation Using Visualization

    可以使用 ::regression plot:: 来以可视化地方式直观的评估 Linear regression ，regression plot 可以展示两个变量之间的关系，和关联系数的强度，以及关联的方向（正关联、负关联）。

    Regression plot 是散点图和直线图的结合。每个点为单个数据，直线图是最后生成的线性回归线。

    ::Residual plot:: 的 y 是 实际值-线性回归预测值 y-yhat，x 是当时使用的 x，也可以把其认为是一组组的误差。根据样本 x 的不断增多，会绘制出一副误差/噪音的图。如果线性回归模型足够准确，误差应该是 y=0 的一条直线。如果图示的点是随机分布，代表线性回归模型具有一定准确性。然而如果图示的点显示出了一定的规律，代表模型不够准确。

    ::distribution plots:: 会以柱形图的方式同时呈现预测值 (y hat) 和实际值 (y) 在不同区间的分布。

    模型样本测试 In-Sample Evaluation

    样本测试可以量化评估模型的准确性和适配性。两种常用的测试方法为： 1. Mean Squared Error (MSE) 即误差的平方总和 / 样本数 2. R-squared (R^2) / coefficient of Determination 目标变量可以用模型来解释的百分比 R^2 = ( 1 - MSE of regression line / MSE of the average of the data) 通常来说范畴为 [0,1]

    蓝线为线性回归模型，红线为指定变量的平均值。 蓝色方块为模型的MSE，红线为平均值的MSE。 由上图可发现蓝色方块显著小于红色方块，证实了模型的有效性。在此情况下，MSE of regression line / MSE of y hat (蓝色方块/红色方块) 会接近于0。

    # MSE
from sklearn.metrics import mean_squared_error
# 第一个值为真实值，第二个值为模型测算出的值
Yhat = lm.predict(X)
mse = mean_squared_error(df['price'], Yhat)

# R-squared
X = df[['highway-mpg']]
Y = df['price']
lm.fit(X,Y)
lm.score(X,Y)

# R2_score
from sklearn.metrics import r2_score
r_squared = r2_score(y,p(x))

    评估和决策制定

    有下列几种方式可以评估模型： 1. 测算的结果是否有依可循 2. 可视化结果展示: residual plot, distribution plot 3. 量化评估分析: MSE 或 R-squared，其中通常来说 MSE 越小越好，r-squared 越大越好 4. 模型间比较评估: SLR vs. MLR, MLR vs PLR

    import numpy as np
# arrange(x,y,z) x为起始点 y为终结点，z为步数
new_input = np.arrange(1,101,1).reshape(-1,1)
# 使用模型
yhat = lm.predict(new_input)

    数值较小的 MSE 是否代表模型更高的适配性？ 答案并不绝对。多维线性回归的MSE通常会小于简单线性回归（MSE for a multiple linear regression model will be smaller than the MSE for a simple linear regression model）。因为多维线性回归纳入了更多的指定变量。

    X variable / independent variable / explanatory variable / predictor variable Y variable / dependent variable / response variable
