    Import and Exporting Data in python

    Import file

    importing 有两个关键的属性需要考虑 * format 文件格式: csv, json, xlsx, hdf * file path 文件路径 * 本地文件 /desktop/mydata.csv * 网络读取 https://archive.edu/autos/imports-85.data

    Importing a csv into python 

    import pandas as pd
url = "<url>"
# 标明文件是否具有 header
df = pd.read_csv(url, header=None)

    常用python方法 * df 返回整张数据表单 * df.head(n) 返回前n行的数据 * df.tail(n) 返回后n行的数据 * df.columns = <headers> 设定列名，其中的形式是列表

    pandas支持读取的数据格式

    Data Format Read Save
    csv pd.read_csv() df.to_csv()
    json pd.read_json() df.to_json()
    excel pd.read_excel() df.to_excel()
    sql pd.read_sql() df.to_sql()

    export file

    path = "desktop/completed.csv"
# index 是表单中的 primary key, 在导出时可以不包括index
df.to_csv(path, index=False)

    Access Data using python

    Python 使用 API 来连接到数据库。

    以上是使用 SQL API 的一张示例图。

    DB API 是python中的标准库，用来连接数据库。且DB API具有可以连接多个数据库的优势。 * connection objects * database connections * manage transactions * cursor objects * database queries * connection methods * cursor() 返回一个新的 cursor object * commit() 确认执行缓存中的操作 * rollback() 撤销在缓存中的操作 * close() 关闭 connection

    from dbmodule import connect
# 创建 connection 对象
conn = connect('databasename', 'username', 'pwd')
# 创建 cursor 对象
cursor = conn.cursor()
# 执行查询语句
cursor.execute("select * from mytable")
result = cursor.fetchall()
# 结束连接
cursor.close()
conn.close()

    Analyze Data in python

    为什么我们需要在导入数据后确定数据的类型呢？ 1. 避免可能出现的 typo 和信息错误 2. 确保和python方法的适配性

    检验数据格式使用 df.dtypes。pandas中的数据格式有 object, int64, float64和datetime64, timedelta[ns]。

    使用df.describe()方法可以返回统计关键数据，如下图所示。如果是针对某一行的，可以使用df[[<col1>,<col2>]].describe()。 Pic 0330-1

    使用df.descirbe(include="all")会返回一张更完整的、包括所有列的统计关键数据，包含 unique, top, freq等。 * unique: 出现了多少个不同的值 * top: 出现频率最高的值，即 mode * freq: top值出现的次数

    使用df.info() 会返回列的数据类型，不是空白数值的计值，dtypes的freq统计，内存使用。

    使用df.replace(<value1>, <value2>) 可以使用value2来替换掉value1。如df.replace('?',np.NaN) 则是使用numpy中的NaN来替换掉?。

    使用df.dropna()可以滤除缺失数据。axis=0 对行进行操作，axis=1对列进行操作。 * df.dropna(how="all") 只丢弃全为缺失值的行 * df.dropna(axis=1) 丢弃有缺失值的列，通常不会操作，因为会缺失一个特征 * df.dropna(axis=1, how="all") 只丢弃全为缺失值的列 * df.dropna(axis=0, subset=["age","sex"] 丢弃age, sex两列中有缺失值的行 * df.dropna(thresh=2) 保留至少有2个非空白值的行
