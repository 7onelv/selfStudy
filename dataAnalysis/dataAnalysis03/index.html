    描述性统计 Descriptive Statistics

    • 描述数据的基本特征
    • 简明扼要地描述数据样本

    描述性统计采用的方法 df.describe() 会返回计数、平均数、最大最小值等，通过使用该方法可以很明晰地洞察数据的分布。

    df.value_counts() 会呈现分类变量的各种数值和计数。分类变量 categorical data 指值是各种分类，如 medium / high / low。

    ::Box Plots 箱形图:: 有点类似于股票中的K线，以可视化的方式呈现最高值、最低值、medium、和各类percentile。在 python 通过调用 sns.boxplot(x="drive-wheels", y="price", data=df) 来实现。

    ==Scatter Plots 散点图== 会将每个数据以一个点的形式呈现出来。散步图可以有效地解释两个变量之间的关系。 * predictor / independent variable on x-axis * target / dependent variables on y-axis

    x = df["engine-size"]
y = df["price"]
plt.scatter(x,y)

plt.title("Engine Size vs Price")
plt.xlabel("Engine Size")
plt.ylabel("Price")

    聚合 GroupBy

    Python 中的聚合可以通过使用 dataframe.Groupby() 方法来实现，其作用对象为分类变量，可以包括一个或多个变量。

    df_test = df[["drive_wheel", "body_style", "price"]]
# group by drive wheel and body style
df_grp = df_test.groupby(["drive_wheel", "body-style"], as_index=False).mean()

    ::pivot 数据透视表:: 在上述结果的基础上，还可以使用 pivot() 方法来将结果可视化。数据透视表会把一个变量放在列上，另一个变量放在行上。

    df_pivot = df_grp.pivot(index="drive-wheels", columns="body-style")

    ::heat map 热力图:: 除数据透视表外，我们也可以使用热力图来分析各个因素对目标变量的影响。

    plt.pcolor(df_pivot, cmap='RdBu')
plt.colorbar()
plt.show()

    数据关联

    Correlation 关联性的定义为在何种程度上变量之间相互依赖。举例来说，吸烟的程度和肺癌具有一定的关联，吸烟越多、患肺癌的几率越高。

    在 python 中可以使用 sns.regplot 来检验两个变量之间的关联。::regplot:: 为 regression plot 的缩写。

    sns.regplot(x="engine-size", y="price", data=df)
plt.ylim(0,)

    Correlation 分为 positive correlation 和 negative correlation。前者随着independent variable 变大 dependent variable 变大，后者则相反。上述图为 positive correlation。

    统计学中的关联 correlation coefficient and p-value

    ::分析关联性的强度 strength of the correlation:: * correlation coefficient 相关系数 * p-value

    当相关系数 * 越接近 1，则其越具有正向的相关关系 * 越接近 -1，则其越具有反向的相关关系 * 越接近0，说明其不具有相关关系

    当 p-value * p-value < 0.001 非常强的结果可确性 * p-value < 0.05 适中的结果可确性 * p-value < 0.1 弱的结果可确性 * p-value > 0.1 结果不具有可确性

    强关联需要满足两个条件。 1. 相关系数非常接近 1 或 -1 2. p-value < 0.001

    在 python 中可以使用一行代码来获取相关系数和p-value。

    pearson_coef, p_value = stats.pearsonr(df['horsepower', df['price'])

    上述的热力图是 17 * 17 的一个方阵，揭示了每个变量之间相互的关系。颜色越深（深红或深蓝）代表其关联系数越高。斜线因为是和自己比较，所以关联系数一定为1，故颜色一定最深。

    卡方检验 Chi-square

    Correlation coefficient 和 p-value 主要服务于 continuous variable。当变量为 categorical variable 即分类变量时，是时候推出 chi-square 卡方检验了。

    卡方检验用来判断在何种程度上观测到的分布符合预先的猜测。卡方检验不会告知变量间存在的是何种关系，但是其会告知变量间是否存在关系。

    下面以一个具体的实例来演示卡方检验的流程： 1. 推出 null hypothesis: 两个变量之间不存在关系 2. 使用 pandas 的 cross-tab 来显示 category variable 之间的分布关系 3. 通过公式计算出chi-square的值，并对照 percentage points of chi-square distribution 来检验其 p-value 4. 证实或回绝 null hypothesis

    在 python 中可使用 scipy 来实现卡方检验，且python会给出具体的p-value值。

    scipy.stats.chi2_contingency(cont_table, correction=True)

    第一个值为chi-square值，第二个为p-value（可以明显看到其小于0.05) ，第三个值为 degree of freedom，第四个为各个分类变量组合的预计值。因为其p-value显著小于0.05，则否决了一开始的null hypothesis，即说明变量之间存在关联性。
